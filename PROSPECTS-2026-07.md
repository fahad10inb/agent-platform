## Find more yourself
- **Bayut** companies by area: bayut.com/companies/dubai/<area>/ (e.g. jumeirah-village-circle-jvc,
  business-bay). Listing count = size proxy (~10–150 = your band, 400+ = large). Call/WhatsApp/Email button.
- **Property Finder** find-broker / find-agent — listing count + WhatsApp button + SuperAgent badge
  (active, lead-dependent = feels the pain).
- **DLD/RERA register** (+ Dubai Brokers app) / **ADREC** (Abu Dhabi) — official names + ORN + license.
- **Instagram** hashtags #dubairealestate #offplandubai #jvcdubai — owner-run = a person in the bio,
  low-thousands–15K followers, WhatsApp link in bio.
- **LinkedIn** search "Founder/MD/Principal broker real estate Dubai"; company filter size 2–10 / 11–50.
